# ReefPulse - Reef Tank Monitor

Track water parameters, diagnose issues, calculate dosing, and monitor trends for your saltwater reef tank.

## Features
- **Water Test Logging** — Track 10 key parameters with coral-type-adjusted ideal ranges
- **AI Diagnostics** — Instant issue detection with severity levels and step-by-step resolution guides
- **Dosing Calculator** — Exact calculations for calcium, alkalinity, and magnesium with unit conversion (tsp/tbsp/ml/g)
- **Trend Charts** — Line graphs for any parameter over time with ideal range visualization
- **Product Recommendations** — Curated suggestions with affiliate-ready links
- **Backlog Support** — Enter past test dates to build historical data

## Tech Stack
- React 18 + Vite
- Recharts for data visualization
- localStorage for data persistence
- No backend required

## Development
```
npm install
npm run dev
```

## Deployment
Push to GitHub and connect to Vercel for instant deployment.
